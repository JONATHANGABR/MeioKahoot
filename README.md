# MeioKahoot
Um kahoot com o tema do meio ambiente para eu fazer e apresentar na minha escola. vou deixar todo o codigo Aqui.

-- Data da Criação: 26/05/2026 : 22:12



## ✅ Requisitos Mínimos

- Ter o **Tailscale** instalado no telefone  
  > É preciso usar essa VPN para poder se conectar ao servidor e jogar.

## 🎮 Como Jogar

1. Instale o **Tailscale** no seu telefone.
2. Conecte-se à VPN.
3. Entre no servidor.
4. Divirta-se!


### Atenção. esse jogo é multiplataforma, pode ser jogado tanto no Computador ou no Celular.
