# MeioKahoot
Um kahoot com o tema do meio ambiente para eu fazer e apresentar na minha escola. vou deixar todo o codigo Aqui.

-- Data da Criação: 26/05/2026 : 22:12

## Para Jogar basta apenas ter os servidor ligado usando NODE.JS
O Servidor é integrado junto com o jogo. para no dia da apresentação eu apenas iniciar tudo no meu computador
Tem varios temas. como "Ciencias, Reciclagem, Água e ETC." no total são quatro tipos de Temas e 100 Questões juntando tudo aquilo.

### Atenção. esse jogo é multiplataforma, pode ser jogado tanto no Computador ou no Celular.
