# MeioKahoot

Um Kahoot com o tema do **meio ambiente** para apresentar na escola. Todo o código está aqui.

-- Data da Criação: 26/05/2026 : 22:12

## Como Jogar
Basta ter o servidor ligado usando **Node.js**. O servidor é integrado com o jogo. Para a apresentação, basta iniciar tudo no computador.

### Atenção
Esse jogo é **multiplataforma**, pode ser jogado tanto no Computador ou no Celular (via navegador).

## Temas e Questões (Atualizado)
Agora são **7 temas** com foco em educação ambiental e **ações práticas**:

- **Ciencias**: Ecologia, biomas, cadeias alimentares, biodiversidade (26 questões)
- **Reciclagem**: Coleta seletiva, 3R's/5R's, compostagem, lixo eletrônico (27 questões)
- **Agua**: Ciclo da água, conservação, poluição, saneamento (27 questões)
- **Clima**: Efeito estufa, aquecimento global, energias limpas, desmatamento (27 questões)
- **Energia**: Renováveis vs não-renováveis, eficiência energética, Brasil renovável (17 questões)
- **Sustentabilidade**: Consumo consciente, desenvolvimento sustentável, pegada ecológica (18 questões)
- **Acoes**: **NOVO** - Focado em "como a gente pode ajudar" o meio ambiente no dia a dia (15 questões)

**Total: 157 questões**

## Estrutura do Projeto
- `Data/Questoes/master.json` - Banco de questões (pode ser editado)
- `Server/` - Código do servidor Node.js
- `Scripts/` - Outros scripts
- `Assets/` - Imagens e recursos

## Dicas para Apresentação
- Escolha temas de acordo com a turma (ex: comece com Ciencias e Reciclagem para mais jovens).
- Incentive discussões após as rodadas: "O que vocês podem fazer em casa para ajudar?"
- Use a categoria **Acoes** para encerrar com motivação e ideias práticas.

Qualquer dúvida ou sugestão, edite as questões no JSON e teste!
